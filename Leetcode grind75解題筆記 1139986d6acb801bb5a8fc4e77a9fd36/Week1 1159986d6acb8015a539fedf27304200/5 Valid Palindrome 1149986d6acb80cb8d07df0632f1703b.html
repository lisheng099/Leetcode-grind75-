5.Valid Palindrome

題目

A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward. Alphanumeric characters include letters and numbers.

Given a string s, return true if it is a palindrome, or false otherwise.

一個字串經過轉化後如果為回文，轉化過程中會將大寫字母改成小寫，並去除所有不是字母跟數字的字元，且從前面讀與後面讀的字元相同。Alphanumeric 包含字母跟數字。

答案

設定左座標為0，設定右座標為字串長度-1。執行while迴圈條件為(left<=right) ，如果字串為空則會直接跳過。檢查左座標指向的字元是否為字母或數字，如果不是則左指標向右移並跳過迴圈內接下來的程式碼，檢查右座標指向的字元是否為字母或數字，如果不是則右指標向左移並跳過迴圈內接下來的程式碼，檢查左右指標指向字元轉為小寫後是否不同，如果不同則回傳false 。

如果歷遍後都沒發生字元不同則回傳true 。

public class Solution {
    public bool IsPalindrome(string s) {
        int left = 0;
        int right = s.Length-1;
        while(left<=right){
            if(!char.IsLetterOrDigit(s[left])){
                left++;
                continue;
            }
            if(!char.IsLetterOrDigit(s[right])){
                right--;
                continue;
            }
            if(char.ToLower(s[left])!=char.ToLower(s[right])){
                return false;
            }else{
                left++;
                right--;
            }
        }
        return true;
    }
}
